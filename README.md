## Посібник користувача 
## для програми "Тетра Квант Симулятор"

Ця програма дозволяє користувачам малювати на полотні з ефектом хвиль та взаємодії з частинками. Ось детальна інструкція щодо використання та налаштування функцій.

## Опис функцій

### Полотно для малювання (Canvas)
Програма має полотно розміром 600x400 пікселів, на якому можна малювати за допомогою миші або сенсорного екрану. Вибір кольору визначає, як буде виглядати лінія, яку малює користувач.

### Кнопки для зміни кольору
Існує кілька кнопок для вибору кольору:

- **Червона**: для малювання червоними лініями.
- **Зелена**: для малювання зеленими лініями.
- **Синя**: для малювання синіми лініями.
- **Помаранчева**: для малювання помаранчевими лініями, які викликають спеціальний ефект.

### Запуск хвилі (Wave)
Кнопка **Запуск хвилі** запускає симуляцію хвиль, яка генерується навколо червоних ліній.

### Відміна останньої лінії (Undo)
Кнопка **Відмінити** дозволяє скасувати останню лінію, яку було проведено.

### Очистити полотно
Кнопка **Очистити** видаляє всі лінії та частинки з полотна.

### Налаштування параметрів частинок
- **Кількість частинок**: Можна змінювати кількість частинок хвилі, вказуючи значення від 1 до 360.
- **Швидкість частинок**: Користувач може налаштувати швидкість руху частинок.

## Опис роботи з програмою

### Малювання
Натискайте на полотно, утримуючи кнопку миші або за допомогою сенсорного екрану, щоб почати малювати. Колір лінії буде змінюватися в залежності від обраної кнопки кольору.

### Запуск хвилі
Після того як ви намалюєте кілька червоних ліній, натисніть кнопку **Запуск хвилі** для активації ефекту хвиль, який поширюється від червоних ліній. Частинки хвилі рухатимуться в різні напрямки.

### Взаємодія з частинками
Помаранчеві частинки, які створюються при малюванні помаранчевих ліній, взаємодіють з частинками хвиль. Вони можуть зіштовхуватися і відштовхуватися від хвильових частинок, створюючи додаткові ефекти.

### Скасування та очищення
Якщо ви хочете скасувати останню лінію, натискайте кнопку **Відмінити**. Щоб очистити все полотно, натисніть **Очистити**.

## Технічні деталі
- Програма використовує **HTML5 Canvas** для малювання.
- Взаємодія частинок з лініями та іншими частинками реалізована за допомогою простих фізичних розрахунків (відбиття, взаємодія з іншими частинками).
- Використовується **JavaScript** для обробки подій миші та сенсорного екрану, а також для симуляції хвиль.

## Як запустити програму
1. Скачайте HTML-файл на ваш комп'ютер.
2. Відкрийте файл за допомогою будь-якого сучасного браузера (наприклад, Google Chrome або Firefox).
3. Почніть малювати на полотні, використовуючи інтерфейс.

## Керування

- **Червоний колір**: натискайте кнопку **Червона**, щоб малювати червоним.
- **Зелений колір**: натискайте кнопку **Зелена**, щоб малювати зеленим.
- **Синій колір**: натискайте кнопку **Синя**, щоб малювати синім.
- **Помаранчевий колір**: натискайте кнопку **Помаранчева**, щоб малювати помаранчевим.
- **Запуск хвилі**: натискайте кнопку **Запуск хвилі**, щоб активувати хвилю.
- **Відмінити**: натискайте кнопку **Відмінити**, щоб скасувати останнє малювання.
- **Очистити**: натискайте кнопку **Очистити**, щоб очистити полотно.

## Параметри

- **Кількість частинок**: ви можете налаштувати кількість частинок хвилі за допомогою відповідного поля.
- **Швидкість частинок**: змінюйте швидкість частинок через текстове поле.

Ця програма дозволяє створювати інтерфейс із художнім ефектом хвильових взаємодій, надаючи користувачам можливість візуалізувати цікаві фізичні ефекти на полотні.
